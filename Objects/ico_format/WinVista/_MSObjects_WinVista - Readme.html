Visual Studio Image Library: Object icons (WinVista style)

Icon art
This filetype may contain more than one image size and/or color depth. They are typically used in controls which offer variations viewing style; a listbox which allows the user to choose between "small icon view" and "large icon view" is an example. The appropriate image size is displayed by the control. It is best to preview these images in the File Explorer; this HTML page will only display the first image in the set contained within the ico file.
